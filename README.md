En este ejercicio creamos una lista de nombres usando ArrayList y se escriben en un fichero de texto llamado nombrex.txt, para ello usamos BufferedWriter junto con FileWriter, recorremos la lista con un bucle for-each y escribiendo cada nombre en una linea diferente gracias a write() y newLine(). Controlamos una posible excepcion con IOException. acabaremos msotrando un mensaje que confirma que los nombres de han guardado correctamente.
